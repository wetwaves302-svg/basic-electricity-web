# 基本電學自學引導 ⚡

> 為臺灣技術型高中（技高）學生設計的互動自學網頁。一步一步學會基本電學。

**開發者：國立花蓮高工 柯貞伊老師**

## 🎯 線上試玩

部署 GitHub Pages 後，可直接訪問：`https://你的帳號.github.io/basic-electricity-web/`

## 📚 功能特色

### 🧮 公式卡（33 個）
- 直流電路、電容、電感、磁學、交流電路五大類
- 每張卡片包含公式、變數說明、使用時機、口訣記憶法
- 可一鍵「📖 看示範」（隨機數值逐步代入）或「✏️ 換你算」（隨機練習題、自動驗算、提供提示）

### 🎮 闖關遊戲（3 關）
- **第 1 關 ‧ 公式配對**：8 對連連看（公式 ↔ 名稱）
- **第 2 關 ‧ 公式填空**：10 題遮一格挑選項
- **第 3 關 ‧ 套公式計算**：5 題隨機數值代入演算
- 含星等（☆☆☆ → ★★★）、徽章、音效、彩帶、總分存檔

### 📊 我的學習儀表板
- **整體理解指數**（0–100 分）：題庫進度 30% ＋ 遊戲星等 30% ＋ 公式正確率 40%
- 6 種等級稱號：🎒準備出發 → 🌱種子發芽 → 🥉起步飛行 → 🥈穩步進步 → 🥇高手在線 → 🏆超強學霸
- 強項 / 弱項自動分析（依據答題紀錄）
- 錯題自動收集，一鍵「再挑戰」或「我會了」

### 🃏 詞彙卡（25 個）
- 電路關鍵詞（KVL、KCL、相量、阻抗、有效值、PF、諧振…）
- 翻面看：一句話定義 / 為什麼 / 例子 / 口訣

### 📚 題庫引導（45 題）
- 交流電路 25 題 + 電容電感磁學 20 題
- 五階段拆解：讀題 → 認公式 → 代入 → 計算 → 驗算
- 「💬 學長口吻」提示、「⚠️ 考爆陷阱」、「🧠 口訣記憶法」

## 🎨 設計理念

- **多階段拆解**：題目拆成 3–5 小步，學生不會被一次性的長篇答案壓垮
- **學長口吻**：用同學間講話的方式解釋，不用學術腔
- **激勵回饋**：答對有音效＋彩帶＋讚美語；答錯有溫和音效＋鼓勵語
- **學習進度記錄**：透過 localStorage 保存在學生自己的裝置上，重開瀏覽器也不會掉
- **手機友善**：響應式設計，分頁列可橫滑，觸控目標大於 44px

## 🔧 技術說明

- 純前端：HTML + CSS + Vanilla JavaScript
- 無依賴：不需安裝任何套件，雙擊 `index.html` 即可使用
- 音效：Web Audio API 即時合成（無外部音檔）
- 視覺特效：CSS 動畫、SVG 儀表板
- 資料儲存：localStorage（個人裝置上）

## 📂 檔案結構

```
basic-electricity-web/
├── index.html       ← 主程式（HTML + CSS + JS）
├── data.js          ← 45 題、25 詞彙、33 公式資料庫
└── README.md        ← 本說明檔
```

## 🚀 部署到 GitHub Pages

1. 在 GitHub 上建立公開 repo
2. 將本資料夾的內容推上去
3. 進 `Settings → Pages`
4. Source 選 `Deploy from a branch`、Branch 選 `main` / `(root)`
5. 等 1–2 分鐘後，GitHub 會給你一個 `https://你的帳號.github.io/repo名稱/` 網址
